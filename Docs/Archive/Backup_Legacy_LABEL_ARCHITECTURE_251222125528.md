# 🏗️ 프로젝트 아키텍처 라벨

> 💡 아래 예시를 참고하여 프로젝트에 맞는 라벨을 추가하세요.

| 라벨 | 경로 | 내용 | 역할 |
|------|------|------|------|
| [APP] | /App/ | 앱 진입점 | @main App 구조체 |
| [VIEWS] | /Views/ | 뷰 폴더 | SwiftUI View 파일 |
| [MODELS] | /Models/ | 모델 폴더 | 데이터 모델 |
| [SERVICES] | /Services/ | 서비스 폴더 | 비즈니스 로직 |
| [RESOURCES] | /Resources/ | 리소스 폴더 | Assets, Localizable |
| [EXTENSIONS] | /Extensions/ | 확장 폴더 | Swift Extensions |

---

> ✏️ 위 예시 행을 수정하거나 새로운 행을 추가하여 사용하세요.
