# 🎨 UI 컴포넌트 라벨

> 💡 아래 예시를 참고하여 프로젝트에 맞는 라벨을 추가하세요.

| 라벨 | 부모 컨테이너 | 위치 | 기능 | 상태 |
|------|--------------|------|------|------|
| [BTN_PRIMARY] | [CONTENT_VIEW] | - | 주요 액션 버튼 | normal/pressed/disabled |
| [TXT_TITLE] | [NAV_BAR] | 중앙 | 화면 제목 | - |
| [LIST_ITEM] | [CONTENT_VIEW] | List 내부 | 목록 아이템 행 | selected/normal |
| [SHEET_MODAL] | [WINDOW_MAIN] | 오버레이 | 모달 시트 | presented/dismissed |
| [TOGGLE_SETTING] | [CONTENT_VIEW] | 설정 행 | 설정 토글 | on/off |

---

> ✏️ 위 예시 행을 수정하거나 새로운 행을 추가하여 사용하세요.
