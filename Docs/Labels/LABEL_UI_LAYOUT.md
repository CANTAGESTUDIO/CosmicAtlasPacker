# 🖥️ UI 레이아웃 라벨

> 💡 아래 예시를 참고하여 프로젝트에 맞는 라벨을 추가하세요.

| 라벨 | 위치 | 크기 | 내용 |
|------|------|------|------|
| [WINDOW_MAIN] | 전체 화면 | 100% | 메인 윈도우 |
| [NAV_BAR] | [WINDOW_MAIN] 상단 | 100% x 44pt | 네비게이션 바 |
| [TAB_BAR] | [WINDOW_MAIN] 하단 | 100% x 49pt | 탭 바 |
| [CONTENT_VIEW] | [WINDOW_MAIN] 중앙 | 나머지 영역 | 메인 콘텐츠 영역 |
| [SIDEBAR] | [WINDOW_MAIN] 좌측 | 250pt | 사이드바 (macOS) |
