# 🔧 상태 메시지 라벨

> 💡 아래 예시를 참고하여 프로젝트에 맞는 라벨을 추가하세요.

| 라벨 | 위치 | 트리거 | 메시지 형식 | 색상 |
|------|------|--------|------------|------|
| [LOG_BUILD] | Xcode Console | 빌드 성공 | ✅ Build Succeeded | 초록 |
| [ERR_COMPILE] | Xcode Console | 컴파일 에러 | ❌ [VIEWS] 컴파일 실패 | 빨강 |
| [WARN_DEPRECATE] | Xcode Console | Deprecated API | ⚠️ iOS 17 deprecated | 주황 |
| [LOG_PREVIEW] | Preview Canvas | 프리뷰 로드 | 📱 Preview 렌더링 완료 | 파랑 |

---

> ✏️ 위 예시 행을 수정하거나 새로운 행을 추가하여 사용하세요.
