```
.label          → 13px, Medium, Primary color

```

## Usage Guidelines

### Hierarchy Rules

1. Use **maximum 3 heading levels** per screen
2. Body text should be **minimum 15px** for readability
3. Caption text should be **minimum 11px**
4. Maintain **consistent line heights** within text blocks

### Accessibility

- Minimum font size: 11px (captions only)
- Recommended minimum: 15px for body text
- Use relative units (rem) in web implementations
- Support Dynamic Type on iOS/macOS
