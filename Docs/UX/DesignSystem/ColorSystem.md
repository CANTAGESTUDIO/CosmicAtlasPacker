## Usage Guidelines

### Do's ✅

- Use semantic colors for states (success, error, warning)
- Maintain 4.5:1 contrast ratio for text
- Use primary colors sparingly for emphasis
- Apply opacity for disabled states (0.5)

### Don'ts ❌

- Don't hardcode hex values in components
- Don't use pure black (#000000) for text
- Don't use colors alone to convey information
- Don't use more than 3 primary color variants per screen
