## Usage Guidelines

### Do's ✅

- Use spacing tokens, never hardcoded values
- Maintain consistent spacing within components
- Use larger spacing to create visual hierarchy
- Group related items with smaller spacing

### Don'ts ❌

- Don't use arbitrary pixel values
- Don't mix spacing systems (4px and 5px base)
- Don't use spacing smaller than 4px
- Don't exceed 80px spacing in normal layouts
